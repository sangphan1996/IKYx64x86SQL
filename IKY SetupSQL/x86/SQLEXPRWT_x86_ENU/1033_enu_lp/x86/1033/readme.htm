Microsoft SQL Server 2008 R2 Readme

01/11/2010 14:02:00

This Readme document contains important information to read before installing Microsoft SQL Server 2008 R2. It describes how to find installation assistance in Books Online, how to find supplemental product information in the Release Notes, and copyright attributions for the product.

This Readme file is available on your installation media at <drive>:\, as well as from the Readme download page.

What's New

For information about the new features in SQL Server 2008 R2, see What's New in SQL Server 2008 R2.

Use Books Online for Installation Assistance

Find Books Online

SQL Server Books Online is the primary documentation for getting assistance with installing SQL Server. You can obtain Books Online in the following ways:

  • Install Books Online by selecting the SQL Server Books Online feature in this release of SQL Server. After installing, update to the latest version, which is available from the Books Online download page.
  • View Books Online on MSDN.
  • Download a stand-alone version from the Books Online download page.

To view the Books Online installation documentation without accessing Books Online, you can open the s10ch_setup.chm file from the installation media or from your computer using the following paths:

  • To open s10ch_setup.chm from the installation media, open <drive>:\[x86] | [x64] | [ia64]\help\1033\s10ch_setup.chm.
  • To open s10ch_setup.chm on your computer, open <drive>:\Program Files\Microsoft SQL Server\100\Setup Bootstrap\SQLServer2008R2\1033_ENU_LP\x86\1033\help\s10ch_setup.chm.

Find Installation Information in Books Online

The following Books Online topics on MSDN contain important information about installing SQL Server 2008 R2. These topics are also in the installed versions of Books Online and in the s10ch_setup.chm file.

Note:
Installing SQL Server 2008 R2 Books Online upgrades your existing SQL Server 2008 version of Books Online. If you upgrade, be aware that the content and help topics now describe tools and features of SQL Server 2008 R2. To reinstall SQL Server 2008 Books Online, you must first uninstall SQL Server 2008 R2 Books Online. Because SQL Server 2008 and SQL Server 2008 R2 share components, you cannot have side-by-side installations of Books Online on the same computer.

Analyze Your Upgrade with Upgrade Advisor

The Upgrade Advisor tool analyzes your current instance of SQL Server and provides important information you should know before upgrading. For more information about SQL Server 2008 R2 Upgrade Advisor, see the Upgrade Advisor Web site.

Read the Release Notes for the Latest Product Information

The Release Notes provide supplemental information about product behaviors that apply only to this release or that are not documented in this release of Books Online. The Release Notes document is available from the Release Notes page.
